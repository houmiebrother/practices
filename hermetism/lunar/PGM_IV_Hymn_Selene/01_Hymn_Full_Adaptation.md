# Гимн к Селене / Гекате

**Источник:** Парижский папирус (P. Bibl. Nat. Suppl. gr. 574, Bl. 31r–31v)
**Строки:** PGM IV.2785–2890
**Датировка:** IV в. н.э.
**Язык:** Греческий (дактилические гексаметры)
**Издание:** Preisendanz, *Papyri Graecae Magicae*, Bd. I (1928/2001), S. 162–165

Гимн-молитва к Селене/Гекате — главный лунный текст корпуса PGM. Обозначен как «молитва к Селене для всякого обряда» (Gebet an Selene, bei jeder Praktik verwertbar) — универсальная инвокация, применимая в любом лунном ритуале. Написан дактилическими гексаметрами (с обычными для магических текстов неточностями метра).

Богиня представлена как синкретическое целое: Селена (полная Луна) = Геката (тёмная Луна / перекрёстки) = Артемида (растущая Луна / охота) = Персефона (подземный мир) = Мойры = Эринии = Дике = Ананке = Фюсис. Все эти «лица» — не перечисление отдельных богинь, а манифестации одной лунной силы.

Структура четырёхчастна:

1. **Гимн-восхваление** (§2786–2849) — эпитеты, облики и атрибуты богини
2. **Вокс магика** (§2849) — палиндромная формула АРАРАХАРА́РА ЭФТИСИКЕ́РЕ (ΑΡΑΡΑΧΑΡΑΡΑ ΗΦΘΙΣΙΚΗΡΕ)
3. **Молитва-прошение** (§2850–2870) — призыв с возжиганием благовоний, просьба об исполнении
4. **Ритуальные указания** (§2871–2890) — приношения, защитный амулет (филактерий)

## Адаптация (современный русский)

*Перевод на основе метрической реконструкции Hymnus 18 (Preisendanz, PGM Bd. II, S. 253–255) с учётом текста папируса (PGM Bd. I), английского (Betz, 1992) и немецкого (Preisendanz) переводов*

### §1. Гимн-восхваление (2786–2849)

Приди ко мне, / любвеобильная, / возлюбленная владычица, / Трёхликая Селена, / и благосклонно внемли / священным песнопениям моим.

Украшение ночи, / юная, / несущая свет смертным, / дитя зари, / на свирепых быках восседающая, / царица, / что мчится в колеснице / одним путём с Ге́лиосом; / в тройственных обликах / трёх Хари́т / танцующая, / со звёздами ликующая.

Ты — Ди́ке / и нити Мойр: / Кло́то и Ла́хесис / и А́тропос.

Трёхглавая, / ты — Персефона, Мегера / и Алле́кто, / многоликая!

Ты вооружаешь руки свои / тёмными, грозными факелами; / ты сотрясаешь гриву / ужасных змей на челе; / ты исторгаешь / рёв быков из уст своих; / чрево твоё покрыто / чешуёй ползучих тварей, / ядоносные ряды змей / на плечах; / стянута по спине / проклятыми цепями.

Кричащая в ночи, / быколикая, / одиночество любящая, / быкоголовая! / Взор у тебя бычий, / голос — собачий; / в голенях своих / ты скрываешь обличья львов; / лодыжка твоя — волчья; / дикоярые псы / милы тебе — / потому и зовут тебя / Гекатой.

Многоимённая, Ме́на, / воздух рассекающая, / подобно стрелометательнице Артемиде!

Четвероликая богиня, / четвероимённая, / четырёх дорог владычица! / Артемида, / Персефона, / охотница на оленей, / в ночи сияющая, / трижды гремящая, / троегласная, / трёхглавая, / трёхимённая Селена, / тризубая, / трёхликая, / трёхвы́йная / и трёх дорог владычица!

Ты, что в тройных корзинах / хранишь неугасимый пламень / и владеешь перепутьями, / и над тремя декадами царствуешь!

Будь милостива ко мне, / к тебе взывающему, / и благосклонно услышь!

Ты, что обширный мир / ночью оберегаешь, / пред кем да́ймоны содрогаются / и бессмертные трепещут!

Мужей прославляющая богиня, / многоимённая, / рождающая прекрасное, / быкоокая, рогатая, / родительница богов и людей, / и Природа — всеобщая Мать!

Ибо ты странствуешь по Олимпу / и обширную бездну / беспредельную обходишь. / Начало и конец — ты, / и ты одна / над всем царствуешь.

Ибо от тебя всё, / и в тебе / всё завершается.

Вечную диадему / носишь ты на висках своих — / узы нерушимые, / неразрешимые — / великого Кро́носа, / и золотой скипетр / держишь в ладонях.

Письмена на скипетре твоём / сам Кро́нос начертал / и дал тебе носить, / дабы всё пребывало нерушимо:

Укротительница, Укрощаемая, / Мужей Укротительница, / Силы Укротительница!

И Хаосом ты правишь.

АРАРАХАРА́РА ЭФТИСИКЕ́РЕ (ΑΡΑΡΑΧΑΡΑΡΑ ΗΦΘΙΣΙΚΗΡΕ).

### §2. Молитва-прошение (2850–2870)

Радуйся, Богиня, / и внемли именам своим!

Возжигаю тебе сие благовоние, / дитя Зевса, / стрелометательница, / небесная, / хранительница гаваней, / гор скиталица / и путей странница!

Подземная и ночная, / богиня Аида / и тьмы, / тихая и ужасающая! / Ты, что трапезничаешь / среди гробниц!

Ночь, / Э́реб, / Хаос обширный — / ибо ты неизбежная Ана́нке! / Ты — Мойра, / ты — Эри́ния, / мука, истребительница, / ты — Ди́ке!

Ке́рбера ты держишь в оковах; / чешуёй змей / покрыта ты — тёмная, / змеевласая / и змееопоясанная!

Кровопийца, / несущая смерть, / порождающая гибель, / сердцеядица, / плотоядица, / пожирающая / безвременно умерших, / воздвигающая вопль, / в исступление ввергающая!

Приди на жертвы мои / и сотвори мне это дело.

### §3. Ритуальные указания (2871–2890)

**Приношения для обряда — эпи́тима (ἐπίθυμα):**
- Для благих дел — стиракс, мирра, шалфей, ладан, фруктовая косточка
- Для вредоносных — магическое вещество от пса и пёстрой козы (или подобным образом — от безвременно умершей девы)

**Защитный амулет — филакте́рион (φυλακτήριον):**

Взять магнитный камень — ли́тос сидери́тес (λίθος σιδηρίτης), на котором вырезана трёхликая Геката:
- **Среднее лицо** — рогатая дева — керасфо́рос парте́нос (κερασφόρος παρθένος)
- **Левое лицо** — пёс — ки́он (κύων)
- **Правое лицо** — коза — экс (αἴξ)

После вырезания — промыть натроном и водой, затем погрузить в кровь человека, погибшего насильственной смертью. Затем совершить приношение пищи — пара́тесис (παράθεσις) — и прочесть тот же гимн во время обряда.

| §1 Гимн | Возвышенный, «Ты» | Гексаметрический гимн; перечисление эпитетов как демонстрация знания |

### Тройная богиня — структура отождествлений

Гимн систематически отождествляет Селену с тремя группами тройственных богинь:

| Тройка | Функция | Лунная фаза |
|---|---|---|
| Три Хариты (Грации) | Красота, радость, танец | Полная Луна |
| Три Мойры (Клото, Лахесис, Атропос) | Судьба: прядение, распределение, неотвратимость | Все фазы (цикличность) |
| Три Эринии (Мегера, Аллекто, [Тисифона/Персефона]) | Месть, возмездие | Тёмная Луна |

Поверх этого — ещё одно тройное отождествление:
- **Артемида** — растущая Луна, дева, охотница
- **Селена** — полная Луна, мать, свет
- **Геката** — убывающая/тёмная Луна, старуха, перекрёстки

И четвертичная схема — те́тра- (τετρα-): четыре лика = четыре фазы Луны или четыре направления перекрёстка.

### Параллельные тексты в PGM

Гимн не изолирован — он имеет прямые параллели внутри PGM IV:

### Сравнение с Систасис Гелиоса (PGM III.494–611)

| | Систасис с Гелиосом | Гимн к Селене |
|---|---|---|
| Назначение | Установление связи — си́стасис (σύστασις) | Универсальная молитва для всех обрядов |
| Регистр | От приказа (§3) к мольбе (§5) к благодарению (§6) | От восхваления к просьбе |
| Божество | Единое (Гелиос = Творец) | Синкретическое (Селена = Геката = Артемида = Мойры = Ананке = Фюсис) |
| Стихотворный размер | Частично гексаметры (§4), остальное — проза | Целиком дактилические гексаметры (с отклонениями) |
| Тональность | Космогоническая, теургическая | Хтоническая, магическая |
| Финал | Молитва Благодарения (герметическая) | Ритуальные инструкции (практические) |

### Филактерий — трёхликая Геката на магнитном камне

Описание амулета уникально для PGM — это одна из немногих детальных инструкций по созданию магического изображения:

- **Материал:** магнитный камень — ли́тос сидери́тес (λίθος σιδηρίτης) — магнетит; магнит как материал, способный «притягивать» силу богини
- **Три лика:** дева (центр) — пёс (лево) — коза (право)
- **Рога на деве:** полумесяц = лунный символ
- **Пёс:** священное животное Гекаты
- **Коза:** жертвенное животное Гекаты (ср. приношения: «пёстрая коза»)
- **Освящение:** натрон + вода (очищение), затем кровь насильственно убитого — биэ́у хэ́ма (βιαίου αἷμα) — связь с хтоническими силами и а́орой (ἄωροι) — безвременно умершими
