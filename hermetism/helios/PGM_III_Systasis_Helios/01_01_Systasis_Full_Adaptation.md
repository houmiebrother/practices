# Систасис с Гелиосом | Σύστασις πρὸς Ἥλιον

**Источник:** Папирус Мимо (Лувр P. 2391, кол. XV-XVIII)
**Строки:** PGM III.494-611
**Датировка:** III-IV вв. н.э.
**Язык:** Греческий
**Издание:** Preisendanz, *Papyri Graecae Magicae*, Bd. I (1928/2001)

Ритуал σύστασις — «соединения», «установления связи» — с Гелиосом. Обозначен как «обряд для всех [целей], для [всего]» (κατὰ πάντων τελετή). Практикующий демонстрирует знание тайных форм и имён Гелиоса по двенадцати часам дня (додекаорос), чем утверждает своё право на связь с богом.

Структура шестичастна:

1. **Воззвание** (§494-500) — призыв Гелиоса от четырёх ветров; заявка на знание его знаков и форм
2. **Додекаорос** (§501-535) — двенадцать часовых форм Гелиоса: облик, дерево, камень, птица, животное, имя
3. **Повелительная формула** (§536-549) — требование исполнения, самоидентификация с Хором (сиротой вдовы Исиды)
4. **Гимн Творцу** (§550-558) — гексаметрический гимн: Бог как основатель мира
5. **Прошение** (§559-590) — просьбы о благах и принятии молитвы
6. **Молитва Благодарения** (§591-609) — Εὐχὴ τῆς Χαριστίας, дошедшая в трёх независимых версиях (PGM III, Асклепий §41b, NHC VI.7)

Папирус сильно повреждён в средней части (часы 7-12 додекаороса, строки 517-535). Восстановление фрагментарное.

*Перевод с греческого оригинала (Preisendanz, PGM Bd. I) с учётом английского (Betz, 1992), немецкого (Preisendanz), латинского (Asclepius §41b), коптского (NHC VI.7) и французского (Festugière) текстов*

### §1. Воззвание (494-500)

[Обряд] для всех [целей], для [всего]. /
О чём бы ты ни просил, / призывай так:

«[Приди,] приди ко мне / от четырёх ветров мира, / воздухоходный, великий Боже!

Услышь меня во всяком обряде, / который я совершаю, / и исполни все прошения / молитвы моей всецело, /

ибо я знаю Твои знаки, / символы и облики, / и кто Ты в каждый час, / и каково имя Твоё.»

### §2. Додекаорос — Двенадцать часов Гелиоса (501-535)

**Час 1 — Детёныш обезьяны (παῖς πιθήκου)**

В первый час Ты имеешь облик / детёныша обезьяны. /
Ты порождаешь дерево — ель; / камень — а́фанос; / птицу — [...]; /
имя Твоё — ФРУВЕ́Р (ΦΡΟΥΕΡ).

**Час 2 — Единорог (μονόκερως)**

Во второй час Ты имеешь облик единорога. /
Ты порождаешь дерево — пе́рсею; / камень — керами́т; / птицу — алуха́кон; / на земле — ихневмо́на; /
имя Твоё — БАЗЕТОФО́Т (ΒΑΖΗΤΟΦΩΘ).

**Час 3 — Кот (αἴλουρος)**

В третий час Ты имеешь облик кота. /
Ты порождаешь дерево — смоковницу; / камень — саму́хос; / птицу — попугая; / на земле — лягушку; /
имя Твоё — АКРАММАХАМАРЕ́Й (ΑΚΡΑΜΜΑΧΑΜΑΡΕΙ).

**Час 4 — Бык (ταῦρος)**

В четвёртый час Ты имеешь облик быка. /
Ты порождаешь дерево — орех; / камень — педэ́рот; / птицу — горлицу; / на земле — быка; /
имя Твоё — ДАМНАМЕНЕ́ВС (ΔΑΜΝΑΜΕΝΕΥΣ).

**Час 5 — Лев (λέων)**

В пятый час Ты имеешь облик льва. /
Ты порождаешь дерево — тёрн; / камень — магни́т; / птицу — [...]; / на земле — крокодила; /
имя Твоё — ФОКЕНГЕПСЕ́ВАРЕТАТУ́МИСОНКТАЙ́КТ (ΦΟΚΕΝΓΕΨΕΥΑΡΕΤΑΘΟΥΜΙΣΟΝΚΤΑΙΚΤ).

**Час 6 — Осёл (ὄνος)**

В шестой час Ты имеешь облик осла. /
Ты порождаешь дерево — колючку; / камень — сапфи́р; / в море — медузу; / на [земле — корову белоло́бую]; /
имя Твоё — ЕИА́У А́КРИ ЛЮ́КС (ΕΙΑΥ ΑΚΡΙ ΛΥΞ).

**Час 7 — [Краб] (κάμαρος?)**

### §3. Повелительная формула (536-549)

Я произнёс Твои знаки и символы! /
Посему, Господи, / сотвори [такое-то дело] по принуждению, / дабы я не потряс небо!

Сотвори [такое-то дело] мне — / Ты, образ и совокупность космоса, / Ты, который, будучи избран, / охранял святилище великого Царя!

Сотвори [такое-то дело] мне, / ключехрани́телю треугольного рая земного, / то есть Царства!

Сотвори [такое-то дело] мне, / чтимому сироте вдовы, — БОЛАТЮРИ́Т (ΒΩΛΑΘΥΡΙΘ), — / дабы не отняли у меня отчизну Господню, /
и да свершится всё благое по повелению!

МАСКЕ́ЛЛИ МАСКЕ́ЛЛО́ ФНУКЕНТАБАО́ АО́РИО ЗА́ГРА РЕСИХТО́Н ХИППО́ХТОН ПЮРОСПАРИПЕГАНЮ́КС КАЙЛА́М ИАЛМИО́ ЛИЛИМУЛЕ́АЛАБАЕНЕРЕДЕ́МУ (ΜΑΣΚΕΛΛΙ ΜΑΣΚΕΛΛΩ ΦΝΟΥΚΕΝΤΑΒΑΩ ΑΩΡΙΩ ΖΑΓΡΑ ΡΗΣΙΧΘΩΝ ΙΠΠΟΧΘΩΝ ΠΥΡΟΣΠΑΡΙΠΗΓΑΝΥΞ ΚΑΙΛΑΜ ΙΑΛΜΙΩ ΛΙΛΙΜΟΥΛΗΑΛΑΒΑΗΝΕΡΕΔΕΜΟΥ)

### §4. Гимн Творцу (550-558)

Приди ко мне / в святом обращении Святого Духа Твоего —

Основателю всего, / богов Боже, Владыко всего, /
устроивший мир / Духом Своим божественным!

Первый Ты из Перворождённого / явился, премудро произрастая / из воды бурной, — / Ты, основавший всё:

бездну, землю, / огонь, воду, воздух и снова эфир, /
реки гремящие, / Луну багряноликую, /
звёзды воздушные, утренние, блуждающие кругами, — /

По помыслам Твоим / они охраняют всё.

### §5. Прошение (559-590)

[Ты,] призывающий [...] /

[...] АМОХЛ [...] ФОДОФ [...] АРФТО [...] /
ПСУФИ́С [...] ОЙОТ О́ФРУЕР ХМЕИ́Б АРПО[ГКНУ́ФИ] БРИ[Н]ТАТЕ́НО[Ф]РИПРОКЮ́ЛМА АРУ́АР ЗАРБАМЕСЕ[Т]КРИФИНИ́ТТУ МИХМУМА́ОФ ИАО́ЛИ, /

прежде чем Ты метнёшь молнию, — /
АИ ХЕСОКИРТАБАОЗА́А ЛЕ АСРИ́С /
о божественноустый, Владыко!

Приди ко мне, Господи, / Ты, то возводящий свет, / то низводящий тьму / силою Своей!

Услышь меня, Господи, / меня, [имярек], / милостиво и радостно / и ко благу, /
от всякой стихии, / от всякого ветра, / в сей день, /
с ликом Твоим светлым, / в сей час, /

ибо я воспеваю святое имя Твоё отовсюду, / Ты, рождённый в каждом человеческом создании, — / вдохни в нас Дух!

Имя Твоё / от правой стороны оси: /
ИАО́ АОИ́ ОАИ́ [ОЮА́] ООООООО ААААААА [...] /
от [левой] стороны оси: /
АО́ АУО́ ИОА́И [П]ИПИ́ ООО ОО ИИИ АУО́ [...] ОАИ.

Приди ко мне / с ликом светлым, / на ложе / по Своей воле избранное, /

даруя мне, [имярек]: /
жизнь, здоровье, спасение, / богатство, благочадие, / знание, чуткий слух, / благосклонность, здравый совет, / славу, память, / милость, благообразие, красоту телесную, — /
красоту в глазах всех людей, / видящих меня!

Ты, слышащий всё от меня, / словам моим даруй убедительность, / великий Боже!

ЕУА́ЕО ИО́ ИАО́ ОА́И О́ТО ЕА́УИ ТАИ́С Е́РХИС /
АУКСА́ХОХ АРСА́МОСИ (ΑΥΞΑΧΟΧ ΑΡΣΑΜΩΣΙ).

Молю Тебя, Владыко, / прими мою литанию, / подношение Тебе, / Ты, повелевший! /
Дабы Ты ныне просветил меня / знанием того, что любезно Тебе, /

и после благого восстановления / материального тела, /
молю, Господи, / прими сие прошение, /
литанию, / предварение, / подношение / духа словесного моего!

И да достигнет оно Тебя, / Владыку всего, / дабы Ты исполнил / все прошения молитвы моей, / Ты, от богов Рождённый!

### §6. Молитва Благодарения (591-609)

Благодарим Тебя всей душой, / сердцем к Тебе простёртым, /

о несказа́нное Имя, / чтимое именем «Бог» / и благословляемое именем «Отец»!

Ты, который ко всем и ко всему / отеческое благоволение, / нежность, любовь и сладостную силу явил, /

даровав нам ум, слово и знание: / ум — дабы Тебя мыслить, / слово — дабы Тебя призывать, / знание — дабы Тебя познавать.

Радуемся, ибо Себя Самого нам Ты явил! /
Радуемся, ибо нас, ещё в телах пребывающих, / обожил Ты знанием Себя Самого!

Благодарность человека к Тебе одна: / величие Твоё познать.

Познали, о человеческой жизни Жизнь! /
Познали, о Лоно всякого знания! /
Познали, о Лоно, Отчей природой чреватое! /
Познали, о вечное пребывание Отца, в Себе Самом всё вынашивающего!

Так, столь великому благу поклонившись, / никакой просьбы не просили мы, /
кроме одной: / благоволи нам сохраниться в знании Твоём / и не уклониться от жизни сей.

## Примечания к переводу

### Структурные решения

| Часть | Регистр | Обоснование |
|---|---|---|
| §1 Воззвание | Прямое обращение, «Ты» | Личный призыв к богу |
| §2 Додекаорос | Ритуальная формула, «Ты» | Демонстрация знания = право на связь |
| §3 Повелительная формула | Императив, «Сотвори» | Магический приказ, не просьба — практик действует с позиции силы |
| §4 Гимн | Возвышенный, гимнический | Гексаметры; ср. CH V и CH XIII |
| §5 Прошение | Молитвенный, «молю» | Переход от приказа к мольбе — смена позиции |
| §6 Молитва | Литургический, «мы» | Коллективное благодарение; самая высокая точка |

### Ключевые переводческие решения

| Греческий | Адаптация | Обоснование |
|---|---|---|
| σύστασις | систасис / соединение | Термин не переводится однозначно: «установление связи», «представление», «соединение». Betz отмечает, что systasis переводится по-разному в зависимости от контекста |
| κατὰ πάντων τελετή | обряд для всех [целей] | τελετή = «посвящение, обряд»; κατὰ πάντων = «для/против всего» |
| μορφὴν ἔχεις | Ты имеешь облик | По образцу PGM IV (ср. PGM_IV_Helios_Consecration.md) |
| γεννᾷς | Ты порождаешь | γεννάω = «рождать, порождать»; солнце «рождает» деревья, камни, животных в каждый час |
| ποίησον τὸ δεῖνα πράγμα | сотвори [такое-то дело] | Формула-заполнитель: практикующий вставляет конкретную просьбу |
| τῷ τῆς χήρας ὀρφανῷ | сироте вдовы | Preisendanz: χήρα = Исида, ὀρφανός = Хор; практик отождествляет себя с Хором, сыном Исиды-вдовы |
| κατατετιμημένῳ | чтимому [сироте] | Dative masc. — согласуется с ὀρφανῷ (сиротой), не с χήρας (вдовой). Перевод спорен: Preisendanz — «verachtet» (презренному), Betz — «honored» (чтимому) |
| κλειδοφύλαξ τοῦ τριγώνου παραδείσου | ключехранитель треугольного рая | Египетский образ: Дельта Нила = «треугольный рай» = Египет |
| ΜΑΣΚΕΛΛΙ ΜΑΣΚΕΛΛΩ... | Маскелли-логос | Стандартная магическая формула PGM, встречается во множестве текстов; функция — принуждение |
| δεῦρό μοι ἐν τῇ ἁγίᾳ σου περιστροφῇ τοῦ ἁγίου πνεύματος | Приди ко мне в святом обращении Святого Духа Твоего | περιστροφή = «вращение, обращение»; генитив «τοῦ ἁγίου πνεύματος» относится к «περιστροφῇ» (обращению), а не к последующему «κτίστα» (Основателю). Betz: «in your holy circuit of the Holy Spirit»; Preisendanz: «im Umschwung deines heiligen Geistes» |
| φὺς εὐμεθόδως | премудро произрастая | εὐμεθόδως = «в стройном порядке, методично»; φύω = «расти, произрастать». Betz: «created carefully»; Preisendanz: «wohl und tüchtig hervorwuchs». В космогоническом контексте «премудро» передаёт упорядоченность божественного творения лучше, чем буквальное «тщательно» |
| ὑσγηνοειδής (σελήνη) | багряноликая (Луна) | ὑσγηνοειδής = «подобная исатису» (красильному растению, дающему багряный/красный цвет). Betz: «red-faced moon»; Preisendanz: «einer Isatispflanze ähnelnd der Mond» |
| ΘΕ[Ι]ΣΤΟΜΑ | о божественноустый | Чтение: θεῖ στόμα = «о божественные уста» (вокатив). Betz читает иначе (BRITHEI STOMA), Preisendanz — как единое слово. Принято чтение θεῖ + στόμα |
| προσύστασις | предварение | προ-σύστασις = «предварительный систасис/соединение»; не путать с «предстоянием» (литургический термин). Betz: «preliminary spell»; Preisendanz: «Prosystasis» |
| Μελιοῦχε / πρωτόγονος | Перворождённый / Мелиух | Мелиух (Μελιοῦχος) — солярный эпитет, букв. «заботящийся о мёде» или «заботящийся о стадах»; встречается в Литургии Митры (PGM IV) |
| ἔμπνευμάτισον ἡμᾶς | вдохни в нас Дух | ἐμπνευματίζω — hapax legomenon (встречается только здесь). Ср. CH XIII.19 с конъектурой Кейля πνευματίζε |
| πνεῦμα λεκτικόν | дух словесный | λεκτικός = «способный говорить»; дух, наделённый речью |
| εὐλογούμενον τῇ τοῦ πατρὸς ὁσιότητι | благословляемое именем «Отец» | Греч. букв. «святостью Отца», но три независимые версии читают иначе: копт. *praised with the name 'Father'*; лат. *religione paterna*; франц. *célébré sous le nom de «Père»*. Трёхверсионный консенсус указывает на значение «благословляемое наименованием "Отец"» |
| ἐπιγλυκυτάτην ἐνέργειαν | сладостную силу | ἐνέργεια = «действенность, сила, активность» (ключевой герметический термин); ἐπιγλυκυτάτη = превосходная степень «сладчайший». «Сладостный» передаёт метафорическую сладость естественнее ЦСЯ-формы «сладчайший»; нарастание списка (благоволение → нежность → любовь → сила) компенсирует отсутствие формальной превосходной степени. Betz: *sweetest power*; Preisendanz: *hoch beglückende Kraft*; лат. *dulcior efficacia*; копт. *any teaching that is sweet and plain* |
| μήτρα κυηφόρε ἐν πατρὸς φυτείᾳ | Лоно, Отчей природой чреватое | κυηφόρος = «чреватый, беременный»; φυτεία = «насаждение, посев». Все версии сохраняют образ чреватости: копт. *womb pregnant with the nature of the Father*; лат. *fecunda praegnatio*; франц. *Matrice grosse de la nature du Père* |
| πατρὸς κυηφοροῦντος αἰώνιος διαμονή | вечное пребывание Отца, в Себе Самом всё вынашивающего | Парадокс: Отец — мужское начало — описан как «беременный» (κυηφοροῦντος). «В Себе Самом всё вынашивающего» делает парадокс видимым без архаичного «чревоносящего». Betz: *pregnant father*; Preisendanz: *der trächtig geht*; копт. *begetting Father* |
| θεῶν γενητέ | от богов Рождённый | Или «Родитель богов» — синтаксис допускает оба чтения |

### Voces magicae

Магические имена транслитерированы кириллицей с ударениями по следующим принципам:
- Основа — греческое произношение (не латинская транслитерация)
- Θ = Т (не Ф), Φ = Ф, Χ = Х, Ψ = ПС, Ξ = КС
- Η = Е/Э (по контексту), Ω = О, ΟΥ = У
- Ударение по позиции в греческом, где определимо
- Греческий оригинал в скобках после каждого имени

### Три дара — трёхчастная формула (§6)

- **Ум** (νοῦς / noûs) — способность мыслить Божественное
- **Слово** (λόγος / lógos) — способность призывать и именовать
- **Знание** (γνῶσις / gnôsis) — способность познавать и быть в связи

Латинская версия (Асклепий) передаёт три дара иначе: conscientia (осознание), ratio (разумение), intellegentia (постижение). Это подтверждает, что речь идёт о трёх ступенях богопознания, а не о трёх отдельных способностях.

## Контекст и параллели

### Додекаорос PGM III vs PGM IV

Две додекаоросные схемы в PGM существенно различаются:

| | PGM III.501-535 | PGM IV.1596-1715 |
|---|---|---|
| Назначение | Систасис (соединение) с Гелиосом | Освящение амулета |
| Формы 12 часов | Обезьяна, единорог, кот, бык, лев, осёл, краб(?), [...], ибис, [сокол?], [...], [...] | Кот, собака, змей, скарабей, осёл, лев, козёл, бык, сокол, павиан, ибис, крокодил |
| Дополнительно | Дерево + камень + птица + животное | Только просьба о даре |
| Завершение | Гимн + Прошение + Молитва Благодарения | «Εἷς Ζεὺς Σάραπις» |

Обе схемы восходят к египетской традиции часовых форм Ра, но представляют разные ветви передачи.

### Молитва Благодарения — три версии

| Версия | Источник | Датировка | Язык |
|---|---|---|---|
| PGM III.591-609 | Папирус Мимо (Лувр P. 2391) | III в. н.э. | Греческий |
| Асклепий §41b | Латинский перевод (приписан Апулею) | IV в. н.э. | Латинский |
| NHC VI.7.63-65 | Кодекс Наг-Хаммади VI | IV в. н.э. | Коптский |

Три версии независимы друг от друга — восходят к общему греческому оригиналу, не дошедшему до нас. Подробный анализ — в файле `Hermetic_Prayer_Thanksgiving.md`.
