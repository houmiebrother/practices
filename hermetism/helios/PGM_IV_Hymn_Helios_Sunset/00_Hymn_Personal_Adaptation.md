# Гимн к Гелиосу на закате

## Версия от 2026-06-11

Несомый дуновением ветров, странствующих в высях, / златовласый Гелиос, / правящий пламенем неутомимого огня!

Ты, что эфирными тропами / великий свод обращаешь, / сам порождающий всё, / что вновь Ты растворяешь. / Ибо от Тебя стихии, / устроенные по Твоим законам, / в четвероповоротном ходе года / весь мир обращают.

Внемли, Блаженный! / Ибо Тебя призываю — / водителя неба / и земли, и Хаоса, и Аида. / И ныне молю Тебя, / Блаженный, / Нетленный, / Владыка мира!

Коли в глубины земли / Ты нисходишь, / к пределам предков, — / испроси для меня исполнить / всё, что в сердце моём и воле моей заключено, / чего желаю в мыслях моих, — / кроткий, милостивый, / не мысля ничего враждебного ко мне.

С седьмого неба / дай мне крепкую благосклонность / у всего рода человеческого и даймонов. / И особенно у женщин, — / сделай меня десятикратно более притягательным. / Сведи меня с той, с которой по пути мне / как можно скорее. / А у мужей сделай меня, — / десятикратно более уважаемым.

И не гневайся / на мои священные заклинания, — / ибо Ты сам повелел, / чтобы люди познали / нити Мойр, — / и это по Твоему наставлению.

Но сохрани / всё тело моё невредимым — / да к свету выйду. / Будь милостив ко мне, / Праотец, / семя мира, / саморождённый!

Призываю имя Твоё, / равное числом Мойрам:
АХАИФОТОТО́ / АИЭ́ / ЭИА́ / ИАЭ́ / ЭИА́ / ТОТО́ / ФИАХА́.
*(ΑΧΑΙΦΩΘΩΘΩ ΑΙΗ ΕΙΑ ΙΑΕ ΕΙΑ ΘΩΘΩ ΦΙΑΧΑ)*

Огненосный, / златосияющий, / светоносный для смертных, / Владыка мира, / да́ймон неусыпного огня, / Нетленный, / златокружный, / чистое сияние лучей / на землю посылающий!
