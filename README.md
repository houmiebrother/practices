# practices

Собрание священных текстов и магических практик разных традиций — герметизм, каббала, православие, планетарная магия и другие. Для каждого источника даётся оригинал, перевод и адаптация для современной практики.

Все адаптации созданы с помощью искусственного интеллекта (обычно Claude).

## Разделы

- **kabbalah** — каббала: Сидур (молитвы)
- **hermetism** — герметизм: греческие магические папирусы (PGM)

### Скоро

- **orthodox** — православие: акафисты, псалмы, каноны, Liber Juratus
- **planets** — планетарная магия: Пикатрикс
- **astral** — астральные практики
- **bardon** — система Франца Бардона
- **love_magic** — любовная магия
- **numbers** — нумерология

Содержимое пополняется постепенно.

## Хотите поучаствовать

Будем рады вкладу. Адаптации делаются по единой методике — промпт лежит в [`prompts/PRAY_ADAPT_PROMPT_v12.md`](prompts/PRAY_ADAPT_PROMPT_v12.md). Берите его, адаптируйте текст и присылайте pull request.
