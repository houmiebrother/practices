# practices

Адаптации священных текстов и магических практик разных традиций.
Для каждого текста — оригинал, перевод и переработанная версия для практики.

## Разделы

- **kabbalah** — каббала: Зоар, Тикуней Зоар, Сефер ха-Разим, Шаар Руах ха-Кодеш, Сидур (молитвы)
- **hermetism** — герметизм: Корпус Герметикум, греческие магические папирусы (PGM)
- **orthodox** — православие: акафисты, псалмы, каноны, Liber Juratus
- **planets** — планетарная магия: Пикатрикс
- **astral** — астральные практики
- **bardon** — система Франца Бардона
- **love_magic** — любовная магия
- **numbers** — нумерология

Содержимое пополняется постепенно.
